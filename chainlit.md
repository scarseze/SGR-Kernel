# SGR Kernel — Multi-Agent Swarm Interface

Welcome to the **SGR Kernel** interactive WebUI.

This interface is powered by a **Multi-Agent Swarm** with specialized agents:

- **RouterAgent** — Routes your request to the right specialist.
- **KnowledgeAgent** — Searches the internal knowledge base (RAG).
- **DataAgent** — Analyzes and visualizes data (CSV, Excel).
- **PeftAgent** — Fine-tunes and configures ML models (PEFTlab).
- **WriterAgent** — Generates documents in GOST/Office format.

Type your task in natural language and the swarm will handle the rest.
